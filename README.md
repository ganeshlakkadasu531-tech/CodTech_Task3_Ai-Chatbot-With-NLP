# CodTech_Task3_Ai-Chatbot-With-NLP

### **AI Chatbot with NLP (Natural Language Processing)**

It involves creating a simple yet intelligent **AI Chatbot** using **Python and NLP libraries** like **NLTK** and **Scikit-learn**.


##  Software Requirements

| Tool                                     | Purpose                         |
| ---------------------------------------- | ------------------------------- |
| **Python (3.8 or higher)**               | To run the chatbot script       |
| **VS Code / PyCharm / Jupyter Notebook** | For coding and running          |
| **Pip**                                  | To install dependencies         |
| **NLTK / Scikit-learn**                  | For Natural Language Processing |
